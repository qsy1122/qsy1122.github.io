# nginx

在mac下

  1. 安装： 先安装(homebrew)[https://brew.sh],再执行brew install nginx
  2. 软链接至nginx： brew link nginx
    • 遇到了没有权限写入的问题 chmod 777 目录名 即可
    1. 修改配置文件
    vim /usr/local/etc/nginx/nginx.conf
  3. 转发配置示例如下:
    location /rest {
    proxy_pass http://192.168.1.143:8080/rest;
}
  4. 中间碰到了su不成功的问题（非必要），需要修改root密码：执行sudo su再执行passwd root 参考资料
